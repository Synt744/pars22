A web-based scraping application for extracting product data from dynamic marketplace websites with an intuitive interface for managing scraping tasks and data.

Core Features:

- Web scraper with bot protection bypass capabilities (user agent spoofing, Cloudflare protection handling, CAPTCHA management)
- Configuration interface for setting target sites, scraping parameters, and data fields to collect
- Product data management dashboard with search and filtering
- Data export functionality in common formats (CSV, JSON)

Visual References:  
Inspired by Octoparse and ParseHub interfaces, known for their user-friendly scraping configuration and data management layouts.

Style Guide:

- Colors: Primary #2563EB (royal blue), Secondary #64748B (slate), Background #F8FAFC (light grey), Text #1E293B (dark slate), Success #22C55E (green), Warning #F59E0B (amber)
- Design: Inter/SF Pro Text fonts, two-panel layout (configuration/results), clean forms with clear sections, responsive data tables, 16px base spacing, light shadows for depth
